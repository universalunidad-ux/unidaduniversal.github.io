Preguntas frecuentes

¿CONTPAQi Colabora tiene costo extra?

No. Colabora es un beneficio incluido sin costo adicional para licencias vigentes de CONTPAQi Nóminas.

¿Cómo acceden los empleados a Colabora?

Vía portal web con usuario/contraseña o app móvil (iOS/Android).

¿La información se actualiza en tiempo real desde Nóminas?

Sí, se sincroniza con la base de datos de CONTPAQi Nóminas.

¿Puedo personalizar el flujo de vacaciones?

Sí, puedes configurar responsables y niveles de autorización.

¿Qué necesito para empezar a usar Colabora?

Tener Nóminas vigente, activar Colabora y dar de alta usuarios.
